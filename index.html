// PROFILE

About

After 16 years in IT — spanning systems analysis, programming, and IT management — I'm now directing that hard-won experience toward cybersecurity. This portfolio documents a hands-on, self-driven lab program designed to build detection and response skills from the ground up, using the same categories of tooling found in real SOC environments.

Each case file in this portfolio mirrors a real SOC workflow stage: detection, incident response, network forensics, and adversary intelligence. Treat it as evidence of initiative and practical hands-on capability going into entry-level SOC and security operations roles.

// INVESTIGATIONS

Case Files

CASE-001 · Detection & Monitoring
SIEM Threat Detection & Dashboard Build
Stand up a working SIEM environment and turn raw security logs into actionable visual monitoring using the Boss of the SOC v2 dataset — an industry-standard simulation of a real breach. Wrote SPL queries, built dashboards from scratch, and rehearsed live demos for non-technical audiences.
Splunk Enterprise BOTSv2 SPL Documented ✓
→ View evidence folder
CASE-002 · Incident Response
Incident Triage, Ticketing & Reporting
Practice the part of SOC work that happens after detection — deciding what matters, tracking it in tickets, and writing it up the way a real analyst hands a case off to the next shift. Structured incident reports covering summary, timeline, IOCs, and remediation.
Ticketing Platform IR Workflow Documented ✓
→ View evidence folder
CASE-003 · Network Forensics
Network Traffic & Packet Analysis
Build the habit of reading raw network traffic instead of trusting summaries. Captured and analyzed live and saved traffic in Wireshark, followed TCP streams end-to-end, and compared baseline "normal" traffic against anomalous captures.
Wireshark PCAP TCP Forensics Documented ✓
→ View evidence folder
CASE-004 · Adversary Intelligence
Honeypot Deployment & Threat Intelligence
Get a firsthand look at what automated attackers actually do by giving them something to attack inside a safely isolated home lab. Deployed a honeypot on Kali Linux, logged credentials tried, commands run, and cross-referenced activity with Wireshark captures of the same traffic.
Kali Linux Honeypot TTP Analysis Documented ✓
→ View evidence folder
