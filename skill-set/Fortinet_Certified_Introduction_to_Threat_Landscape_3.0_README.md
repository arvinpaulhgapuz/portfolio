# Fortinet Certified Introduction to Threat Landscape 3.0

## 📖 Overview

This repository contains notes, learning materials, and practical insights gained from the Fortinet Certified Fundamentals (FCF) – Introduction to Threat Landscape 3.0 course. The course provides foundational cybersecurity knowledge by exploring modern cyber threats, attack methodologies, and defensive strategies used to protect organizations from evolving security risks.

---

## 🎯 Learning Objectives

By completing this course, learners will be able to:

- Understand the current cybersecurity threat landscape.
- Identify common cyber threats and threat actors.
- Recognize attack vectors used in modern cyberattacks.
- Understand the Cyber Kill Chain framework.
- Learn the fundamentals of threat detection and mitigation.
- Identify security technologies used to defend networks and systems.
- Develop cybersecurity awareness and risk management skills.

## 🛠 Topics Covered

### Threat Landscape Fundamentals
- Evolution of cyber threats
- Modern attack trends
- Cybercrime ecosystem
- Threat intelligence basics

### Threat Actors
- Cybercriminals
- Nation-state actors
- Hacktivists
- Insider threats
- Organized cybercrime groups

### Common Cyber Threats
- Malware
- Ransomware
- Spyware
- Trojans
- Worms
- Botnets
- Phishing attacks

### Attack Techniques
- Social engineering
- Credential theft
- Exploitation of vulnerabilities
- Distributed Denial-of-Service (DDoS)
- Advanced Persistent Threats (APTs)

### Cyber Kill Chain
1. Reconnaissance
2. Weaponization
3. Delivery
4. Exploitation
5. Installation
6. Command and Control (C2)
7. Actions on Objectives

### Security Controls

- Antivirus and Anti-malware
- Intrusion Detection Systems (IDS)
- Intrusion Prevention Systems (IPS)
- Security Information and Event Management (SIEM)
- Sandboxing
- Secure Email Gateways

## 🔍 Key Concepts Learned

### Confidentiality, Integrity, and Availability (CIA Triad)
- Protecting sensitive information
- Maintaining data accuracy
- Ensuring system availability

### Defense in Depth
A layered security approach that combines multiple security controls to reduce organizational risk.

### Threat Intelligence
The collection and analysis of threat data used to identify, prevent, and respond to cyber threats.

### Security Awareness
Understanding how human behavior impacts cybersecurity and how organizations can reduce risk through education and training.
